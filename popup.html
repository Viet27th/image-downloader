- Nhập Selector trên website. Nếu không sẽ mặc định tìm kiếm ảnh trên cả trang.

- Quy tắc nhập "Selector":

  • Nếu là class: thêm dấu chấm "." trước tên class
  • Nếu là id: thêm dấu thăng "#" trước tên id

- Nhập "File name". Nếu không nhập, file name sẽ mặc định theo thứ tự file bắt đầu từ 0

- Quy tắc nhập "File name":

  • File name là tiền tố, ví dụ, nếu bạn nhập file name là "picture_" thì các file download về sẽ có tên như sau: "picture_1", "picture_2", ...

- Nhập "Folder name" là tên của thư mục sẽ chứa những ảnh được tải về
